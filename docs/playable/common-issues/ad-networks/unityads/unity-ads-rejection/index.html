Unity Ads Rejection for Automatic redirect from playable to the app store

This may be caused by having the Open Store With End Card option enabled when the last interaction prior to the end card is over 1 second apart.

Possible solution:

  • Create a duplicate of the creative and set this new version to have the Open Store With End Card option disabled, you can label this version to indicate it is specially for Unity Ads. Reupload to Unity using this new version.
