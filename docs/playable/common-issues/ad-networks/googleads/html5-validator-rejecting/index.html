HTML5 Validator rejecting build for size of 150kb exceeded

This is likely occurring if you are using the old HTML5 Validator (HTML5 Validator), which was for Google Ads builds smaller than 150KB when the 5MB beta for Google had yet to be released.

Possible solution:

  • Test via uploading to the Google Ads website directly ([Google Ads - Get More Customers & Generate Leads with Online Ads](https://ads.google.com/
