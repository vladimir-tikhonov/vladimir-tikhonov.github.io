Exported builds failing without errors stuck on Preloader screen or failing with TypeError

The issue involves an exported build that fails silently without errors, or it gets stuck on the Preloader screen, or it fails with an error: TypeError: Cannot read properties of undefined (reading ‘slice’)

Possible solution:

  • Add a regular cube to the scene, move it off screen and rebuild for Creative Suite.
