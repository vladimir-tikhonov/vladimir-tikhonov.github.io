TypeError Cannot read properties of TextMeshPro null reading length

This problem may occur when the Parse Escape Characters option in the TextMeshPro component is enabled. This feature is currently not supported in Luna.

Possible solution:

  • Disable the Parse Escape Characters option in the TMP component.
