Skybox is not rendered when using the Luna shader variants collection

It is likely that the required skybox shaders, or their keywords, do not exist in the Luna.shadervariants file.

Possible solution:

  • Check which shader the skybox is using and assign it to Luna.shadervariants. Please also make sure to include any additional keywords that are used (most commonly, it is <no keywords>).
