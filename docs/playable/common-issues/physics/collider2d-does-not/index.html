Collider2D does not contain a definition for ClosestPoint warning during build

Users may encounter this issue when trying to use the ClosestPoint function in Collider2D which is currently not unsupported by Luna.

Possible solution:

  • Switch to a 3D collider if you wish to use the ClosestPoint function.
