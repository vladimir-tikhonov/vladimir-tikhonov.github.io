Ad Network Settings

In the Playable Plugin Unity plugin interface, a number of ad network settings which help prepare your playable ad for use on the ad networks. Navigate to the Settings section and the Ad Network tab.

Global Settings​

important

The App store links previously available in the Ad Network Settings have been removed. You can now configure them within the Application Settings in the Creative Suite front-end.

Application Name​

The name of your application, which will be used in the Preloader displayed before the ad has loaded.

Orientation​

Some ad networks (TikTok and Tencent) require that you define whether your playable supports responsive or fixed orientation.

Facebook​

Asset ID (optional)​

The unique ID of your asset.

Package type (ZIP / Inline)​

Whether the build for Facebook is provided as a ZIP (5Mb limit) or a single inlined HTML file (2Mb limit).

ironSource​

API type​

As outlined here, the API which will be used in the playable ad.

It is recommended to leave this setting as Legacy (DAPI). Changing the API type expands to include additional options.

  • API mode: pa (playable) or iec (interactive end card)
  • Version code (please consult ironSource documentation)
  • Application Genre

Package type (Inline / CDN)​

Whether the build for ironSource is provided as a single inlined HTML file, or used a CDN to deliver the assets.

TikTok & Tencent​

Orientation​

Both TikTok and Tencent require that you define whether your playable supports responsive or fixed orientation.

TikTok​

Supported Languages​

TikTok requires that you specify which languages your playable will support.
