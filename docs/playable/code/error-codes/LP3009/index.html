  • 3009 - API not supported

API not supported

Message​

You are using a function/component which are currently not supported in the Playable Plugin.

Explanation​

Some Unity APIs are not supported in Luna, either because of a known bug or because of limitations of ad networks.

Steps to fix​

Please remove the line of code, usually there is a workaround to be found. Please contact your Technical Solutions Engineer for support on this.
